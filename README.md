# blur-avatar
自动批量模糊化微信聊天记录头像（for Jonns）

# 速度
经过测试，处理160张图片仅需要**2分钟**。

# 效果
有 `直接黑边` 与 `画条纹`两种效果。颜色默认是灰色，可以自定义。
